# C++ Projects Collection

A collection of 13 complete C++ applications covering OOP, systems programming, networking, algorithms, data structures, and games.

## Projects

| # | Project | Category | Description |
|---|---------|----------|-------------|

| 3 | [Mini Shell](mini-shell/) | Systems | A basic Unix shell with pipes and redirection |
| 4 | [Memory Pool Allocator](memory-pool/) | Systems | Custom memory allocator with pool-based allocation |
| 5 | [HTTP Server](http-server/) | Networking | Minimal HTTP/1.1 server that serves static files |
| 6 | [JSON Parser](json-parser/) | Data | Parse JSON strings into a C++ data structure |

| 11 | [Markdown Parser](markdown-parser/) | CLI Tool | Parse Markdown to HTML |
| 12 | [Student Database](student-database/) | OOP | CRUD operations on student records with file I/O |
| 13 | [Expression Calculator](expression-calculator/) | Algorithms | Evaluate math expressions with operator precedence |

## Building

Each project includes a `Makefile`. To build any project:

```bash
cd <project-folder>
make
./<executable>
```

All projects use C++17 and compile with `g++` or `clang++`.
