# Bank Management System

A command-line banking application with persistent account storage, supporting multiple accounts and full transaction tracking.

## Features

- Create accounts with initial deposits
- Deposit and withdraw funds
- Transfer money between accounts
- View account balance
- Full transaction history per account
- List all accounts
- File-based persistence (`accounts.dat`)

## Build

```bash
make
```

## Usage

```bash
./bank
```

Follow the interactive menu to manage accounts.

## Clean

```bash
make clean
```
